Droidweb

Android app to wrap mobile optmized website

Download Download View Droidweb on GitHub

This is an Android app to wrap mobile website.

This is useful if you have a website which is already optimized for mobile users and all you want is a simple app wrapper that displays the website.

You have to just change URL in the source to have your own website URL.
